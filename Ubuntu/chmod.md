# chmod

## Permissions
### Symbols (exclude special symbol as "S")
* d = Directory
* r = Read
* W = Write
* x = Execute

### Table
|Digit | Read Human | Binary|
|------|------------|-------|
|7     |rwx         |111    |

* E.g
  * chmod 777
    rwx   |rwx   |rwx
    Owner |Group | Others
