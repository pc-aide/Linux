# mke2fs

## Name
* mke2fs - create an ext2/ext3/ext4 file system

## Switch
* -t --fs Type

## Syntax
mke2fs -t \<typeVolume\> \<path.device\>

## E.g.
````Bash
mke2fs -t ext4 /dev/xvdb
````
