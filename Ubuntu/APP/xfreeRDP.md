# xfreeRDP

## Syntax

## Switch
* -size: 
  * Default (implicit) : 1024x768
  * 1440x900

## Example
* EC2 Windows Server:
````bash
xfreerdp /v:3.96.146.170 /size:1024x768  /u:Administrator /monitors:2 #rdp-WS2019
````
